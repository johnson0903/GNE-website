盛通電子庫存呆滯料管理部門簡介

公司庫存的問題，跟如同人體內膽固醇一樣。好的膽固醇，讓人體健康，有好的庫存才能即時滿足客戶訂單，贏得商機。但不好的庫存，造成公司資金積壓與無形成本浪費，如同人體積存不好的膽固醇一樣成動脈硬化，血液循環不佳。

盛通電子呆滯料庫存管理部門成立已經超過十年，成立目的就是希望協助解決電子製造與電子元件通路商庫存積壓問題，盡量有效資源再利用。 盛通庫存呆滯料管理部，收購IC電子元件庫存經驗已超過十年，從一開始收購工廠庫存，陸續增加收購代理商、原廠庫存、同行的庫存電子元件。 收購地點也由台灣，慢慢擴展至香港、中國、韓國、日本、歐美地區。在客戶的支持肯定與團對成員的努力，收購庫存電子元件業績一直持續成長。

將電子元件直接報廢或許是個選項，但如能在報廢前幫每個零件找尋再利用機會，就可以對減少地球資源消耗多盡一份心力，如果貴公司有庫存電子元件轉售需求，請將貴公司電子元件庫存清單email至 oes@gneic.com，我們會盡全力用合理價格、彈性的交易方式收購貴公司庫存。

常見問題

1. 如何委託盛通電子收購庫存？

您只需將貴司盤點後確認要轉售庫存電子元件清單email oes@gneic.com，盛通電子庫存呆滯料管理部門估價人員，會立即為您估價。

2. 給盛通電子庫存電子元件清單需要那些欄位？

*原廠料號、*品牌、*數量、Date Code (*為必要欄位)

3. 盛通電子報價需要多久時間？

盛通在收到貴司庫存電子元件清單後，48小時內提供報價。

4. 盛通電子如何付款？

盛通使用現金或電匯方式，支付貨款給您。您不需擔心交易不到貨款風險，收到貨款，再出貨即可。

5. 收購庫存電子元件有那些？

盛通電子收購電子元件產品以主動IC電子元件最擅長，如二三極管、CPU、IC、Chipset等。其他收購產包含被動元件電容、電阻。機構件如連接器等。

6. 確認交易後，如何將庫存電子元件交給盛通？

貨在台灣地區：盛通將自行派員取件。
大陸退港物件：您指定的香港取貨Forwarder貨倉。
內地交易：您指定地方。

7. 報廢庫存比轉售庫存好？

請參閱我們更詳細說。

8. 前公司被併購，剩下庫存，無法開立發票，可以用個人名義轉賣給盛通?

盛通是一家會計師實地查帳公司，所以交易都需要有正式憑證，如果個人會依國稅局規定，用”一時個人貿易所得”方式，收購您的庫存。
